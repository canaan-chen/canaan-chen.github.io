数字货币投资复盘

2021-03-25

这个将作为我的每日复盘合集，每天复盘今日的投资情况和思考未来的投资策略，希望每天都有新的发现

3月25日 — BTC交割在即，市场抛售恐慌

  1. 今天受BTC季度交割的缘故，整体大盘从58000跌到50000，昨天首富喊话BTC可以支付特斯拉，但是效果只能持续半天，之后便开始大跌。下午3点左右首富又喊话支持Defi，但是效果一般，上涨稍许后就又开始下跌趋势。
  2. 受sushi解锁的缘故，sushi今日继续下跌15%，到达14u左右，我在16u左右入场，加了三倍杠杆，损失了将近50%，期间略微加仓。问题操作：没有在考究市场的情况下直接在16u入场，抄底失误，依旧是入场太着急的缘故，后续操作应该依旧会持有
  3. 今日的明星股Fil，应我之前的说法，长期看好，在大盘大跌的时候依旧能上涨。个人不足操作：昨天在高点就抛售了，以为今日会跌下来重新建仓，事实上依旧徘徊在高位，其实昨天的撤仓操作没有问题，因为自己对Fil的了解还是略片面，谨慎没有问题，不过看到它今天的表现，我觉得是一个很有潜力的货币。
  4. 今日略微进入THETA，建仓10%，说实话我对THETA的了解不是特别多，只是最近看到惊人涨势了解了一下，发现是做区块链数字媒体的，想做去中心化的YouTube，打算周末抽时间仔细研究一下。
  5. 整体信息面：今日土耳其的里拉崩盘，有人分析是因为美国过分印钞票防水的缘故，市场资本有开始回撤的迹象，似乎在应对美国的加息策略，不过鲍威尔又表明会继续维持2%的通货膨胀率，不会改变，因此个人觉得大概率还是会保持降息策略，周末打算研究这方面的联动影响，出一篇相关的文章。
  6. 操作计划：静等明天交割情况，在BTC没有站稳53000的情况下不大仓进入，小仓位进一些明星货币

3月26日 — BTC短线回暖，整体大盘上涨

  1. 投的几个数字货币都有较大的反弹上涨，其中Sushi的反弹上涨最猛，这也是可以理解的，因为前期跌得太厉害了，自然会有较大的反弹空间。主要原因是之前关于解锁的消息已经被市场消化地差不多了，此外sushi发布一键杠杆的产品，属于利好消息，至于具体有什么用其实我不是特别理解。可以逐步逢低加仓，个人感觉现在sushi价格属于偏低的，保持30%左右仓位就行。
  2. 今天依旧要提一下Filecoin，不光顶住了大盘下跌的影响，还反向增长，突破了100美元，我稍微搜了一下发现今天Fil社区有关于一个FIP-14的提案，要求将v1证明扇区的周期延长到540天，这其实是一个正向的利好消息，因为说明项目在往好的方向发展，这个提案是为了增大Fil存储的安全性。
  3. 今天还有一个利好消息面是国家十四五规划里将区块链定位发展方向，其地位和人工智能、互联网一样重要。可以开始布局一些靠谱的国产币，具体哪个币我就不说了。
  4. 今天大盘挺坚挺的，没有昨天的颓势，波动恢复到了53000的点，短期内应该会在这个点震荡，但是如果有一个利好消息，整个趋势会直接上杨，预测会触顶63000，然后站稳55000

3月27日 - 疯狂的市场下如何hold住自我

早间新闻记录：

  1. ETH为什么推迟opm的发布，可能原因是因为近期关于EIP 提案的协议没有完全落实到底，部分矿工反对可能导致链分叉的情况，因此在这个时间段不适宜推出layer 2的方案。
  2. 近期多家基金在SEC提交发行比特币ETF，整个SEC的决策开始不是禁止ETF，而是选择哪家首发，但是因为3月24号大幅度回调的原因，最近应该不会立马发出。预测如果为了发布ETF，机构可能会稳定上涨趋势，在一个合适的时间段发行，然后瞬间大盘爆拉，之后再缓慢回调。

今日复盘：

  1. 今日明星依旧是Fil，从昨晚到今天涨幅有快40%，最高直逼138美元，随后缓慢降落，在125美元区间震荡。我是看着这个货币不断上涨的，说实在我也犯了新手容易犯的问题就是追涨。老实说如果这个币一直涨上去我觉得还是很可怕的，幸好有所回落开始震荡，说明需要大家冷静下来逐步建仓。
  2. 昨天提及要布局国内的币，之所以没有说明是哪一个币，是因为害怕自己的判断失误，不过今天看涨势确实符合预期，也就不怕说了，这个币是CFX，一个国产币运用树状分支的方式来加快区块链效率的技术。今天CFX冲高涨幅15%，之后逐步回落到7%，还是比较有前景的，尤其是应对当下国内重视区块链技术发展的大背景。
  3. 在写复盘的时候，突然看到消息ZKsync 5月份上线evm，8月份上线主网，随后发现ZKswap暴涨。这其实是一件比较难分析的事，首先为什么ZKS会暴涨？1. 可能一些新手被名字误导，认为ZKS就是ZKsync，所以投资错误（ZKsync是Mattter Lab的产品，目前没有发币，如果利好ZKsync的话可以买CRV）； 2. ZKsync上线缓慢，近期好用的layer2 可能就只是ZKS，所以也可以理解为一个ZKS的利好消息。
  4. 关于行为金融学：（1）过度自信。人类容易过于相信自己对股票价值的判断能力，并且依据自己的判断进行交易，经常会过度操作导致亏损。（2）市场并不是完全理性的，有时候会出现误差，它是所有人情绪的一个集合。

3月28日 - 国际局势动荡，币圈该何去何从

  1. 今日重点关注的新闻是中国和伊朗签署战略合作协议，不得不说这是牵制美国的一步重棋，继中美协商没有达成，并且美国在协商上表现不尊重姿态之后，美国还就新疆问题不断造谣踩中方的底线，因此在这个矛盾升级的阶段，中国境内先发生对相关外国品牌的抵制行为，之后外交部发言人直接在新闻发布会上发出关于新疆的视频以批判美国的造谣行为，直到今天中伊建立战略协议并且同意人民币购买石油，这其实是一步狠棋，直接赤裸裸挑战美元的地位。接下去就要看美国的态度了，如果美国一直表示高调示威，不肯用谦卑的态度来换回中美合作的话，那接下去整个世界的经济将会开始动荡，这一次大国之间的博弈愈发精彩。
  2. 动荡局势对币圈有什么影响呢？事实上，这一次的动荡，会影响美国的债券利率变化，如果再没有外资购买美国国债，那么美国国债的利率继续上升会加剧美国的债务，当然美国也可以狠下心来硬扛着用增发美元的方式来处理国债，这样相当于让美国的通货膨胀转嫁到全球市场，影响美国的信誉。还有一个手段是加息，提早戳破美国市场的泡沫，不过介于目前的经济复苏以及拜登刚上任的情况，加息手段估计不会立马使用。整体来说我认为币圈依旧是一个可以避险的投资手段。
  3. 最后要说的是关于Sushi的一个声明，它解释说四月份的token不是一次性的释放，而是claim的释放，可以把它作为一个利好消息来处理，毕竟因为四月释放的问题，sushi一直在跌，目前的点有可能触及到了一个成本问题，所以为了维持价位，sushi官方会释放利好来拉动价位，短期应该会有一个不错的回调。毕竟sushi也是DEX的二把手。
  4. 今天的失误操作是没有给crv加仓，因为其实昨天我已经分析出来crv这波利好，但是看到价格迟迟没有涨动就又把资金撤出，今天早上看到crv一路涨幅20%，就有一丝丝遗憾，不过即使遗憾但是我知道了两点：1. 信息的传递是有延迟性的；2. 我对币圈的形式判断还是有正确性的。

3月29日 - 多方利好，依旧需要谨慎投资

  1. 今日有多边利好消息，比如拜登又要签发三万亿刺激法案，visa支持数字货币支付等等，其实我主要还是想分析拜登的刺激法案，因为其实从我之前的分析可以知道，如果美国想化解自己的债务危机，要么主动和中国合作，要么就是继续印钞票，显然他选择第二个，因此可以预测未来一段时间内通货膨胀会继续下去，依旧看好BTC的涨势
  2. 今日操作分析：在听到DOT要解锁的时候，小仓位预测dot会短跌，事实上并没有，市场上整体的向上趋势覆盖了消息面的影响。今日CFX猛涨后又滑落，具体原因是CFX要上币安，但是后来可能大家发现只能用bsc上的cfx，然后就抛售了，老实说，我依旧觉得cfx是一个好项目，可以长期持有，当然对于那些只是看着名字买的人那我就不建议持有了，因为没有信念的话在币圈是很难持有的。另外在前期uni下跌的时候没有加仓是一个遗憾的地方，要知道uni即使v3令人失望，但是其依旧是DEX的龙头项目。
  3. 最后聊一个我朋友的故事，我有一个朋友他和我一块儿进的币圈，他喜欢玩杠杆和合约，而我则玩现货，他几次预测都很准，并且每次发给我的收益都很令我羡慕，他用6000玩到了2w，而我则还处于亏损中，但是他今天跟我说他爆仓了，准备退圈一段时间，我说为啥不和我一样玩现货，他说因为回不去了，现货的收益已经满足不了他内心的狂野。我也曾一度想玩合约，但是我发现光玩现货就已经花费了很多经历，可能只有拥有足够多的经验才有底气玩合约。

3月30日 - 多处平台声明愿意用数字货币结算

  1. 今日是一个多重利好消息发布的一天，最为重要的莫过于paypal 未来支持用数字货币结算，其实伴随着各种大公司愿意用数字货币结算的消息，数字货币的被认可度也不断的提高。截至目前BTC24小时涨幅1.4%，最高触及59000的高压线。现在的大盘在多重利好下有望触及60000的水准。但是依旧得注意的是目前都是靠消息面拉动大盘，一旦消息面被消化完可能就会大跌，目前下一个重要的消息点就是SEC同意比特币的ETF。
  2. 今天写复盘的时间略晚是因为在忙着给FIL加仓，刚刚在fil横盘了一周左右突然被拉盘，因此在仓位上调整，更加侧重fil。现在发现其实个人投资没必要撒网太多的币种，会容易导致没有时间分析每一个货币。
  3. 今日操作：今日整体人比较浮躁，操作仓位略显随意。今日sushi发布公告后，市场反应依旧缓慢，16u是压力点，不容易突破，毕竟还是有代币要被放出来。整体仓位向fil倾斜。
  4. 今日明星板块：平台币、借贷币种

3月31日 - Fil to the Moon

  1. 今日大盘整体涨势喜人，近期fil依旧是明星，根据非小号数据，目前Fil已经超越Uni成为第八大货币，长期来看依旧看涨。今天主要聊聊一个话题关于我是如何投资fil的，其实最初我并不知道fil，当我知道fil的时候它已经录入灰度，涨幅到一个难以上车的高度，但是一个“双花”乌龙契机告诉我这是一个上车的机会，因为一般闹出乌龙事件的项目都会在暴跌之后上涨，所以暴跌的时候就是建仓的好时机，之后我又开始补功课了解fil的机制，认定这是一个又前景的项目后开始长期定投。没上车的朋友也不用担心，因为币圈好的项目有很多，相比于割了这个，投了那个的行为，不如等待好机会好项目再上车。
  2. 近期我想做一个关于小白投资的教学文章，会涵盖币圈的一系列知识，作为自己的一个韭菜成长记录，希望到时候能给大家带来启发。
  3. 小插曲关于Sushi，今天Sushi放出一个Q2的计划图，理论上应该是利好的，但是市场反应惨淡，项目挑三月最后一天的今天放出消息，应该是为了缓和4.1解锁的市场恐慌情绪，只是效果并不明显，sushi的建仓点可以再等候。
  4. 近期也会再出一篇关于美债的分析文章。

4月1日 — 愚人节的狂欢

  1. 愚人节特辑：今天是愚人节，各大币种老板也开始开玩笑，首先是孙哥开玩笑说Ripple和SEC和解，XRP短线上涨，然后首富说要带狗狗币去月球，因此Doge也暴涨一番。当然今日最大的消息是币安要投资Mask，Mask直接突破到22u，币圈真的是很有意思，因为你完全没法预测到下一个会暴涨的项目。
  2. 接下去的操作指导：可以关注分布式存储、web3.0、以及稳定币相关的项目。可以逐步布局Defi等待六月份的Layer2 上线，准备近期也会出一篇关于Defi相关的投资汇总。
  3. 近期利好分布式存储，除了FIL之外还有分布式存储的其他项目：STORJ（以太坊）、BTT（波场）、CRU（波卡）、AR（波卡）、BLZ（COSMOS生态）
  4. 最后想说两点。一是投资的心态，我最近听了朋友的一个故事，说是他在拉斯维加斯玩老虎机，用2美元赚到了200美元，很开心，但是当他发现坐他隔壁的老哥用2美元挣了2000美元的时候，他就变得不开心了，似乎自己吃亏了，于是接着下注最后连那200美元也没有了。其实这个故事很能反应大家在投资时的心态，有时候听到周边的朋友加大杠杆挣了很多钱，然后心里很痒觉得自己如果怎样怎样就也能挣更多，其实这种不平衡的心态是人性的一个弱点，大家总是容易选择性地忽视利益背后的风险，而把目光聚焦到收益上。二是我最近听了一个教授的话，很有感触，就是非常感激这个五彩斑斓的世界，区块链世界真的很丰富多彩，有很多值得玩味的地方，也有很多很棒的想法和idea，如果真的想着价值投资的话，那赚钱真的是一件很容易的事。
